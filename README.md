# Twitter Login/Register Application with Jetpack Compose

This Android Studio project showcases a basic login and registration system utilizing Jetpack Compose and Twitter API for authentication.

## Key Features

- **Twitter Login**: Enables users to sign in to the application using their Twitter credentials.
- **Registration**: Allows new users to create accounts by connecting their Twitter profiles.
- **Jetpack Compose UI**: Utilizes Jetpack Compose for building a modern and declarative UI.
- **OAuth Authentication**: Implements Twitter API's OAuth authentication for secure user authentication and registration.
